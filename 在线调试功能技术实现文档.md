# 在线调试台（Sandbox）技术实现文档

> 本文档详细描述了基于 Docusaurus 3 的"在线调试"功能的完整技术实现，供 AI 在其他项目中复刻使用。

---

## 一、功能概述

在线调试台是一个嵌入在 Docusaurus 文档站点中的浏览器端 API Playground，提供以下能力：

1. **实时连接后端服务**：通过 SDK 连接云渲染服务器或 WebSocket API 服务
2. **代码编辑与执行**：CodeMirror 编辑器 + Ctrl+Enter 即时执行
3. **API 示例导航**：左侧树形分类浏览，点击即填入/执行
4. **参数可视化调节**：lil-gui 风格浮动面板，拖滑块实时改代码并重跑
5. **AI 代码生成**：集成 13 个 LLM 提供商，自然语言生成 SDK 代码
6. **文档到调试台桥接**：文档代码块"亲自试一试"按钮一键带入
7. **代码分享**：URL hash 编码代码，一键复制分享

---

## 二、技术栈

| 层次 | 技术 |
|------|------|
| 框架 | Docusaurus 3（React 18） |
| 路由 | 文件系统路由：`src/pages/sandbox.js` → `/sandbox` |
| 编辑器 | `@uiw/react-codemirror` + `@codemirror/lang-javascript` + `@codemirror/theme-one-dark` |
| AI 调用 | 浏览器直连各 LLM 提供商 REST API（SSE 流式） |
| 状态管理 | React useState/useRef/useCallback（无 Redux） |
| 数据生成 | Python 脚本从 API 文档 Markdown 自动生成 JS 数据文件 |
| SDK | 外部 SDK（`ac.min.js`）运行时动态加载 |

### 4.3 参数面板（src/components/ParamPanel.js）

#### 4.3.1 功能描述

浮动在视频流右上角的 lil-gui 风格面板。解析编辑器中的 `fdapi.*` API 调用，提取参数对象，结合文档元数据渲染 GUI 控件。用户调节参数后，将新值写回代码字符串并（可选）实时重执行。

#### 4.3.2 代码解析流程

```
编辑器代码 → findFirstCall() → 提取 {ns, method, argsText}
                                    ↓
                            parseObject() → [{key, valueText, comment, start, end}]
                                    ↓
                            buildControl() → 合并 PARAM_META 元数据 → 控件描述
                                    ↓
                            渲染 Control 组件（slider/color/bool/enum/number/vector/string）
```

**findPrimaryCall** — 正则找首个 `fdapi.*` 调用：

```js
function findPrimaryCall(code) {
  const re = /(?:fdapi|fdplayer|api)\.([\w$]+)(?:\.([\w$]+))?\s*\(/g;
  let m;
  while ((m = re.exec(code))) {
    const open = re.lastIndex - 1;
    const close = matchParen(code, open);  // 括号匹配
    if (close < 0) continue;
    const inner = code.slice(open + 1, close).trim();
    if (!inner) continue;  // 无参方法跳过
    const ns = m[2] ? m[1] : null;
    const method = m[2] || m[1];
    return { ns, method, open, close, argsText: code.slice(open + 1, close) };
  }
  return null;
}
```

**matchParen** — 括号匹配（处理字符串/转义/嵌套）：

```js
function matchParen(code, open) {
  let depth = 0, str = null;
  for (let i = open; i < code.length; i++) {
    const c = code[i];
    if (str) { if (c === '\\') { i++; continue; } if (c === str) str = null; continue; }
    if (c === '"' || c === "'" || c === '`') { str = c; continue; }
    if (c === '(') depth++;
    else if (c === ')') { depth--; if (depth === 0) return i; }
  }
  return -1;
}
```


#### 4.3.3 控件类型

| kind | 渲染 | 说明 |
|------|------|------|
| `bool` | `<input type="checkbox">` | 布尔值 |
| `enum` | `<select>` | 枚举下拉，选项来自文档参数表 |
| `slider` | `<input type="range">` + number box | 有 min/max/step 的数值 |
| `number` | `<input type="number">` | 无范围的数值 |
| `color` | `<input type="color">` + hex 文本框 | 支持 #hex / Color.Name / [r,g,b] |
| `vector` | N 个 `<input type="number">` | 2-4 维向量 |
| `string` | `<input type="text">` | 字符串 |

防止 AI 幻觉生成不存在的 API：

```js
// 构建白名单（来自 api-completions.js + api_examples.js）
function buildCompletionApis() {
  const s = new Set();
  for (const ns of Object.keys(API_COMPLETIONS.ns)) {
    s.add('fdapi.' + ns);
    for (const m of API_COMPLETIONS.ns[ns]) s.add('fdapi.' + ns + '.' + m.label);
  }
  for (const m of API_COMPLETIONS.root) s.add('fdapi.' + m.label);
  return s;
}

// 校验 AI 返回的代码
function findInvalidApis(responseText, whitelist) {
  const calls = responseText.match(/fdapi(?:\.\w+)+/g) || [];
  return [...new Set(calls.filter(c => !whitelist.has(c)))];
}

// 使用：流式结束后校验
const invalidApis = findInvalidApis(full, validApis);
if (invalidApis.length > 0) {
  // 替换消息为错误提示，拦截该响应
}
```

### 4.5 文档"亲自试一试"按钮（src/theme/CodeBlock/index.js）

Docusaurus 主题覆写（theme swizzle）：

```js
import OriginalCodeBlock from '@theme-original/CodeBlock';

const RUNNABLE_LANGS = ['js', 'javascript', 'jsx', 'ts', 'typescript'];

export default function CodeBlock(props) {
  const sandboxUrl = useBaseUrl('/sandbox');
  const lang = (props.className || '').replace(/.*language-([\w-]+).*/, '$1') || props.language || '';
  const code = typeof props.children === 'string' ? props.children : '';
  const runnable = RUNNABLE_LANGS.includes(lang) && code.trim().length > 0;

  if (!runnable) return <OriginalCodeBlock {...props} />;

  const runInSandbox = () => {
    localStorage.setItem('SbPendingCode', code.trim());
    window.open(sandboxUrl, '_blank', 'noopener');
  };

  return (
    <div style={{ position: 'relative' }} className="runnable-codeblock">
      <OriginalCodeBlock {...props} />
      <button onClick={runInSandbox} /* 样式略 */>
        ▶ 亲自试一试
      </button>
    </div>
  );
}
```

关键点：只给 `js/javascript/jsx/ts/typescript` 语言的非空代码块添加按钮。

**输出 1**：`static/dts-sdk.d.ts` — TypeScript 声明文件

```ts
declare namespace fdapi {

}
declare const api: typeof fdapi;
declare function log(msg: any, noLineBreak?: boolean, color?: string): void;
declare function sleep(ms: number): Promise<void>;
```

**输出 2**：`src/data/api-completions.js` — 补全数据

```js
export const API_COMPLETIONS = {
```

## 七、核心交互流程

### 7.1 文档 → 调试台

```
用户在文档页看到 JS 代码块
  → 点击"亲自试一试"按钮
  → localStorage.setItem('SbPendingCode', code)
  → window.open('/sandbox', '_blank')
  → sandbox 页面 useEffect 读取 SbPendingCode
  → setCode(pending)
  → 若"立即执行"开启且 fdapi 已就绪：
     → pendingAutoRunRef.current = true
     → onReady 回调中 setTimeout(() => eval(code), 300)
```

### 7.2 AI 生成 → 执行

```
用户在 AI 助手输入"开启降雨"
  → send() 构建 messages + system prompt（含 buildApiRef 注入 weather 签名）
  → fetch SSE 流式请求
  → 实时更新消息气泡
  → 流式结束后 findInvalidApis() 白名单校验
  → 用户点击"插入编辑器"
  → setCode(aiGeneratedCode)
  → 用户按 Ctrl+Enter 或开启"立即执行"自动运行
```

### 7.3 参数调节 → 实时反馈

```
用户在编辑器写 fdapi.weather.setRainParam({ strength: 0.5 })
  → ParamPanel 解析代码，找到 weather.setRainParam 调用
  → parseObject 提取 { strength: 0.5 }
  → buildControl 结合 PARAM_META 推断为 slider（min=0, max=1）
  → 渲染滑块控件
  → 用户拖动滑块到 0.8
  → serialize → "0.8"
  → applyChange 替换代码中 0.5 → 0.8
  → onTweak → setCode + eval（防抖 250ms）
  → 视频流中雨效变大
```

## 八、安全注意事项

1. **XSS 防护**：SDK onLog 回调内容必须 escapeHtml 后再写入 DOM
2. **API Key 安全**：仅存 localStorage，浏览器直连 AI 提供商，不经中间服务器
3. **代码执行**：使用 `window.eval()` 执行用户代码，等同于浏览器控制台，无额外沙箱
4. **分享链接**：代码编码在 URL hash 中，不会发送到服务器
