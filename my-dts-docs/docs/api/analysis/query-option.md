# QueryOption

数据库/属性查询的选项配置对象，用于设置查询字段、过滤条件与范围，作为查询接口的参数。
## 业务场景 Skill

> 本节面向 AI 与业务人员，说明本对象在数字孪生业务中的定位与典型用法。

- **功能介绍**：QueryOption 是数据库/属性查询的选项配置对象，用于设置查询字段、过滤条件与范围，作为查询接口的参数。
- **别名 / 不同行业叫法**：查询选项 / 查询条件 / 属性查询参数 / 筛选条件。
- **适用行业**：GIS 与智慧城市、资产与设备管理、智慧水利、能源管网
- **使用场景**：
  - 按属性条件筛选图层要素并高亮定位
  - 空间范围 + 属性条件联合查询设备/管线
  - 构建专题图、报表所需的查询条件
- **注意事项**：
  - 本身是参数配置对象，需配合具体查询接口使用
  - 字段须与数据源 schema 对应
  - 查询范围/结果集过大会影响响应性能

## 构造函数

```js
new QueryOption()
```
