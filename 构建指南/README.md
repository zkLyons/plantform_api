## 🎯 阶段概览

### 阶段一：项目初始化与基础搭建
**目标**：搭建项目框架，实现基础页面路由和布局
- 创建Vite + React项目
- 配置基础路由
- 实现基础布局组件
- 加载3个测试文档

### 阶段二：文档渲染系统
**目标**：实现Markdown渲染、代码高亮、文档展示
- 实现Markdown渲染引擎
- 实现代码块组件（高亮+复制）
- 实现API参数表格渲染
- 实现目录导航（TOC）
- 批量导入文档

### 阶段三：导航与搜索系统
**目标**：实现侧边栏导航、目录导航、搜索功能
- 实现完整的侧边栏导航
- 实现面包屑导航
- 实现全文搜索
- 实现API方法搜索

### 阶段四：在线调试系统
**目标**：实现代码沙箱、控制台输出、实时执行
- 实现代码沙箱执行环境
- 实现控制台输出捕获
- 实现错误捕获和显示
- 实现SDK的fdapi对象注入

### 阶段五：AI代码助手
**目标**：实现智能代码生成、API验证、多模型支持
- 实现AI代码助手组件
- 实现多AI提供商集成
- 实现API白名单验证
- 实现流式输出

### 阶段六：性能优化与部署
**目标**：优化性能、响应式设计、部署上线
- 实现主题切换（暗色/亮色）
- 实现响应式布局
- 实现性能优化
- 部署到生产环境
